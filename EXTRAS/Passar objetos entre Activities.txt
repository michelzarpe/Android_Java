Muitas vezes pode ser necessário passar objetos entre activities, para fazer isso existem algumas maneiras, segue o passo a passo de umas das maneiras de como fazer:

1 - Sua classe terá que implementar a interface "Serializable", note que coloquei um método getDadosUsuario para recuperar os dados, mas poderá utilizar os getter e setters normalmente.

import java.io.Serializable;
 
public class Usuario implements Serializable {
 
    String usuario;
    String senha;
 
    public User(String usuarioParam, String senhaParam) {
        usuario = usuarioParam;
        senha = senhaParam;
    }
 
    public String getDadosUsuario(){
        return "Usuario: " + usuario + " Senha: " + senha;
    }
 
}


2 - Para passas os dados com um evento de clique no botão:
import android.content.Intent;
import android.support.v7.app.AppCompatActivity;
import android.os.Bundle;
import android.view.View;
import android.widget.Button;
 
public class MainActivity extends AppCompatActivity {
 
    private Button botao;
 
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
 
        botao = (Button) findViewById(R.id.passarDados);
 
        botao.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
 
                Intent intent = new Intent(MainActivity.this, DetalhesActivity.class);
                Usuario usuario = new Usuario("Jamilton", "1234");
                intent.putExtra("objeto", usuario );
                startActivity(intent);
 
            }
        });
 
    }
 
}

3 - Recuperar o objeto passado utilizando o método "getSerializableExtra" e fazendo uma conversão para o tipo do objeto "(Usuario)"
import android.content.Intent;
import android.support.v7.app.AppCompatActivity;
import android.os.Bundle;
import android.widget.TextView;
 
public class DetalhesActivity extends AppCompatActivity {
 
    private TextView texto;
 
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_detalhes);
 
        texto = (TextView) findViewById(R.id.texto);
 
        Intent i = getIntent();
        Usuario usuario = (Usuario) i.getSerializableExtra("objeto");
 
        texto.setText( usuario.getDadosUsuario() );
 
    }
}
